HTML (Hypertext Markup Language)


HTML es un lenguaje que sirve para estructurar la web, con el podemos agregar textos, imagenes y configurar la posicion de los elementos en las pagina web.
(Cabe aclarar que HTML no es un lenguaje de programaci칩n).



Importancia de nombrar un archivo html como "index.html":

El index es lo primero que llama o que carga el servidor al momento de ingresar a una pagina.
Por ejemplo en una pagina tenemos la seccion de "Contactar", "Productos", "Servicios", etc...
Sin embargo al momento de buscar la pagina, lo que cargar치 pirmero al ingresar a la misma sera el "index."

Como cuando ingresamos a Facebook, carga el index, no la seccion de notificaciones, ni los mensajes, ni el perfil, sino el index que es como la pagina de inicio. (Aunque en Facebook la configuracion es distinta por lo tanto su modo de operar es diferente, asi que es como ejemplo).



Especificaciones y elementos:
-En HTML todo se hace con codigo.

Etiquetas
Se utilizan "etiquetas" que son como objetos, es decir que al crear una etiqueta es como si se estuviera creando un objeto.
Las "etiquetas" tambien son como cajas en las que se ingresa informaci칩n e instrucciones.
*Existen etiquetas para cada cosa, como para poner >> Botones, espacios, texto, campo de texto, negrita, titulo, seleccionar color, etc.

Atributos
Los "atributos" son como las propiedades o caracteristicas del objeto y los atributos tienen un "valor".
*Pueden haber varios "atributos" en una etiqueta.

Valor
Como su nombre lo indica, se refiere al "valor" o la especificacion de un "atributo".
*El valor siempre se pone entre comillas "value". ------- Ej: color="verde".
